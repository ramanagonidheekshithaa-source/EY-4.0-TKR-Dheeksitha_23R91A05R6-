Left Column

This is the left column content area. You can add text, images, or other HTML elements here. On wider screens, it appears side-by-side with the right column.

The left column background is a Green to visually distinguish it.

Right Column

This is the right column content area. It also supports any HTML content. When viewed on smaller screens, this column will stack below the left column.

The right column background is a Yellow Green color for differentiation.
